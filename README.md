**Service AMG**  
**Тема: WEB**  
**Портал Service AMG для записи автомобиля на ремонт**  
**На сайте можно:**  
  &nbsp;&nbsp;&nbsp;•	Без регистрации:  
    &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;o	Просматривать главную страницу с описанием услуг и контактами.  
    &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;o	Читать прайс-лист (примерные цены на работы).  
    &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;o	Смотреть примеры выполненных ремонтов.  
    &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;o	Перейти на страницу входа / регистрации.  
  &nbsp;&nbsp;&nbsp;•	С регистрацией:  
    &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;o	Изменить имя, номер телефона, e-mail.  
    &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;o	Сменить пароль.  
    &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;o	Удалить аккаунт.  
    &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;o	Добавить новый автомобиль (марка, модель, год, госномер).  
    &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;o	Просмотреть список своих машин.  
    &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;o	Отредактировать или удалить автомобиль.  
    &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;o	Создать новую заявку  
    &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;o	Выбрать автомобиль из списка.  
    &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;o	Описать проблему.  
    &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;o	Указать желаемую дату и время.  
    &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;o	Видеть текущий статус каждой заявки:  
**Реализация**  
  &nbsp;&nbsp;&nbsp;Сайт создан на Flask и sqlalchemy. Реквизиты существующих пользователей находятся в фале passwords.txt  
Service AMG - это удобный портал для записи автомобиля на ремонт. Всё начинается с регистрации: нужно указать имя, номер телефона и e-mail. После подтверждения данных вы попадаете в личный кабинет. Там вы выбираете неисправность (двигатель, ходовая, электрика), дату и время.
