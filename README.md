**Service AMG**  
**Тема: WEB**  
**Портал Service AMG для записи автомобиля на ремонт**
**На сайте можно:**
  •	Без регистрации:
    o	Просматривать главную страницу с описанием услуг и контактами.
    o	Читать прайс-лист (примерные цены на работы).
    o	Смотреть примеры выполненных ремонтов.
    o	Перейти на страницу входа / регистрации.
  •	С регистрацией:
    o	Изменить имя, номер телефона, e-mail.
    o	Сменить пароль.
    o	Удалить аккаунт.
    o	Добавить новый автомобиль (марка, модель, год, госномер).
    o	Просмотреть список своих машин.
    o	Отредактировать или удалить автомобиль.
    o	Создать новую заявку
    o	Выбрать автомобиль из списка.
    o	Описать проблему.
    o	Указать желаемую дату и время.
    o	Видеть текущий статус каждой заявки:
**Реализация**
  Сайт создан на Flask и sqlalchemy. Реквизиты существующих пользователей находятся в фале passwords.txt
Service AMG - это удобный портал для записи автомобиля на ремонт. Всё начинается с регистрации: нужно указать имя, номер телефона и e-mail. После подтверждения данных вы попадаете в личный кабинет. Там вы выбираете неисправность (двигатель, ходовая, электрика), дату и время.
